## 🚀 Featured Project: YouTube Automation Engine

[![yt-auto-rhymes](https://img.shields.io/badge/Python-100%25-3776ab?style=flat-square)](https://github.com/sinhalbarot05/yt-auto-rhymes)

I built a **fully serverless, zero-budget Python pipeline** that autonomously generates, renders, and publishes 3D Pixar-style Hindi toddler Shorts **4 times a day** to [Hindi Masti Rhymes](https://www.youtube.com/@HindiMastiRhymes).

### Key Features:
- 🧠 **3-Tier LLM Hydra** - Fallback system (OpenAI → Groq → WaveSpeed)
- 🎨 **4-Layer Asset Generation** - Robust image pipeline with multiple providers
- 🎬 **Automated Video Production** - MoviePy assembly with dynamic effects
- 📡 **YouTube API Integration** - SEO-optimized uploads via chunked resumable transfers
- ⚙️ **CI/CD on GitHub Actions** - Runs 4 scheduled broadcasts daily (< 5 min per run)

**Tech Stack:** Python • OpenAI API • MoviePy • YouTube API • GitHub Actions • Edge-TTS
